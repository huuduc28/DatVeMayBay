* Đồ án xây dựng website phục vụ cho hãng hàng không *

* Cách cài đặt:
- Tải xampp tại: https://www.apachefriends.org/download.html
- Copy toàn bộ project vào thư mục htdocs. Thường đường dẫn sẽ là C:\xampp\htdocs

* Cách cấu hình:
- Tại folder database của project, có 1 file tên là air_ticket_management.sql. 
- Start Apache và MySQL ở xampp control panel
- Bấm vào mục Nhập (import) và trỏ tới file air_ticket_management.sql phía trên 
Cấu hình database:
- Bấm vào "Các tài khoản người dùng", lấy các thông tin host, username, password
- Vào project, model/connection.php, cấu hình theo đúng các thông tin ở trên

*Run project:
- Mở xampp control panel lên và start Apache và MySQL, project sẽ tự động load vào file index.php là trang chủ của website
